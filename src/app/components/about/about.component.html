Network Device Management System
Primary Features
  1. Add device
  2. Edit device
  3. Device detail page
  4. Device search functionality
  5. Dashboard statics
  6. Search device
 Routes
  • /dashboard : analytics and device list
  • /devices: list of all the present devices
  • /device/:id : device complete information
  • /about : basic information about the project
 Services
  • Manage devices service: Keeps and maintains the list of devices. (CRUD)
  • Snackbar message service: To display and notify user about updates
  • Modal service: Manages dynamic components i.e. Edit Device Modal, Add Device Modal
 Custom Directive
  • Angular material elevation Directive: Changes elevation (box-shadow) on hover
 Primary Components
  • Components
  • Device list
  • Device detail
  • Search device
  • Navbar
  • About
  • Page not found
 Modal components Components used as a pop over (dialogs)
  • Add device component
  • Edit device component
  • Confirmation box
 Model used (object class) Device:
  • name : string
  • brand : string
  • version : string | number
  • active : boolean = true
  • description? : string
  • type? : any
  • index? : number
